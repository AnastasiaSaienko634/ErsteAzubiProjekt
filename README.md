# Azubi Helper 🚀

Das ist ein interaktives Lern- und Hilfetool für Auszubildende. Es nutzt Künstliche Intelligenz, um Fragen zu beantworten, Lernstoff zu erklären und Unterstützung bei Hausaufgaben zu bieten.

---

Das Projekt ist über Vercel deployed und direkt im Browser nutzbar.
## Funktionen 🌟

- **Fragen stellen:** Nutzer können Fragen zu verschiedenen Themen eingeben.
- **Lernhilfe:** Erklärungen, Zusammenfassungen und Tipps zum Lernstoff.
- **Hausaufgaben-Hilfe:** Unterstützung bei Aufgaben, ohne direkte Lösungen vorzugeben.
- **Pagination** and a "Load More" button to load additional camper cards.
- **AI-Integration:** Antworten werden KI-gestützt über Mistral AI + Make.com generiert.
